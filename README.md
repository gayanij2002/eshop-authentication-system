# E-Shop Authentication System

This project is a user authentication system for an e-commerce platform. It includes user registration and login functionalities with client-side and server-side validation.

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL Workbench
* XAMPP
* HeidiSQL

## Features

* User registration
* User login
* Form validation
* Database integration
* Responsive user interface

This project demonstrates my understanding of front-end and back-end web development, including form handling, validation, and database connectivity.
